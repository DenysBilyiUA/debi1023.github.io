Indikerar fetstil Indikerar innehållet innanför taggen ska betonas med emfas. I Indikerar text som på något vis skiljer sig från övrig text, t.ex. tankar eller utländska ord. SMALL Finstilt innehåll, t.exx. copyright-information.
STRONG Innehållet i taggen har stark betydelse. SUB Nedsänkt text. SUP Upphöjd text.
INS Markerar att text lagts till i ett dokument, t.ex. för att indikera att det redigerats.
DEL Markerar att text tagits bort ifrån ett dokument, t.ex. för att indikera att det redigerats. Ofta visas detta genom överstruken text.
MARK Markerar text på något vis, t.ex. för att markera upp sökta ord i en text
